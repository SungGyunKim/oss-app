# WBS (Work Breakdown Structure) — OSSTEM Desktop App

## 1. 인증

### 1.1 로그인
- SSO 로그인 페이지를 통한 로그인
- 로그인 성공 시 메인 화면 자동 전환

### 1.2 로그아웃
- 트레이 메뉴에서 로그아웃
- 서버 로그아웃 + 쿠키 제거 + 로그인 화면 복귀

### 1.3 세션 만료
- 웹 서비스에서 세션 만료 감지 시 자동 로그아웃

## 2. 메인 화면

### 2.1 사이드바 메뉴
- BOOK (예약)
- CRM (고객 관리)
- POST (채팅) — 기본 페이지
- JOB (채용)
- MY PAGE (내 정보)

### 2.2 콘텐츠 영역
- 사이드바 메뉴 선택에 따라 웹 페이지 전환

## 3. 채팅 알림

### 3.1 실시간 메시지 수신
- 로그인 시 WebSocket 자동 연결
- 채팅 메시지 실시간 수신

### 3.2 토스트 알림
- 화면 우측 하단에 알림 팝업 표시 (발신자, 시간, 메시지 미리보기)
- 5초 후 자동 닫힘
- 클릭 시 해당 채팅방 열기

### 3.3 채팅방 창
- 알림 클릭 또는 웹 서비스 요청으로 개별 채팅방 창 열기
- 동일 채팅방 중복 창 방지

## 4. 시스템 트레이

### 4.1 트레이 아이콘
- 시스템 트레이에 앱 아이콘 상주
- 더블클릭 시 앱 열기

### 4.2 트레이 메뉴 (비로그인)
- 로그인 / 종료

### 4.3 트레이 메뉴 (로그인)
- 앱 열기 / 로그아웃 / 종료

## 5. 사용자 프로필

### 5.1 프로필 조회
- 로그인 시 사용자 정보 자동 조회 (이름, 회원 유형 등)


## 6. 앱 동작

### 6.1 트레이 상주
- 창을 닫아도 앱 종료되지 않고 트레이에 유지

### 6.2 단일 인스턴스
- 앱 중복 실행 방지, 기존 창 포커스

## 7. 환경 및 배포

### 7.1 환경 분리
- 스테이지 / 운영 환경 별도 관리

### 7.2 패키징
- Windows 설치 파일 (exe) / macOS 설치 파일 (dmg) 빌드
